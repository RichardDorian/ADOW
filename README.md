# ADOW

A Decently Optimized Windows

# What does it do?

- Enables `Ultimate Performance` power plan
- Adds `Take Ownership` to the context menu
- Bypass Windows 11 requirements check
- Allows offline installation
- Disables account lockout policy
- Disables Windows Defender
- Disables System Protection
- Enables long paths
- Allows PowerShell scripts to run
- Disables app suggestions & widgets
- Disables telemetry & data collection
- Removes most bloatware (see list [here](docs/bloatware.md))
- Disables last access timestamp
- Disables mouse acceleration
- Enables dark mode by default
